# 용어 사전 (Glossary)

모든 문서·UI 문구·커밋 메시지는 아래 표준어만 사용한다. 금지어 발견 시 표준어로 교체한다.

## 확정 용어

| 표준어 | 정의 | 금지어 | 코드 식별자 |
|--------|------|--------|------------|
| 경험 남기기 | 방문자가 평판 키워드+피드백을 제출하는 행위(시트). 하나의 제출이 두 산출물로 나뉜다 | 리뷰 남기기, 한마디 남기기 | `ExperienceBottomSheet`, `submitExperience()` |
| 평판 키워드 | 경험 남기기에서 선택된 키워드의 누적 집계. 상위 5개가 프로필에 표시 | 경험 키워드 | `reputationKeywords` |
| 피드백 | 경험 남기기에서 함께 남긴 메시지. 평판 키워드와 별도로 목록 관리 | 방명록, 한마디, 리뷰, 경험 | `guestbook` → 통합 리네이밍 예정 |
| 피드백 요청 | 지인에게 피드백 작성을 요청하는 기능 | — | `onRequestFeedback` |
| 회원가입 | 계정 생성 절차: 소셜 로그인 → 약관 동의 → 본인인증 | 온보딩 (이 구간에 사용 금지) | `Step1Login`, `StepTermsAgreement`, `Step2Verify` |
| 온보딩 | 회원가입 완료 후 프로필 초기 설정: 기본정보 → 프로필 → SNS → 연락 수단 → 하이라이트 → 자기소개 | 회원가입 (이 구간에 사용 금지) | `Step2BasicInfo` ~ `Step9Complete` |

## 폐기 개념

| 폐기 | 사유 |
|------|------|
| 방명록 | 별도 기능 아님 — 피드백의 옛 이름 |
| 경험 섹션 (네트워크 탭의 키워드+메시지 묶음 목록) | 기능 아님 — 평판 키워드 집계와 피드백 목록으로 분리 표시 |

## 미확정 (혼용 중 — 확정 필요)

| 후보 | 혼용 위치 |
|------|----------|
| 관계 탭 / 네트워크 탭 / 평판 탭 | docs는 "관계·평판", 코드 라벨은 `NETWORK` |
| 라이프 탭 / 바이브 탭 | docs·코드 id는 `life`/`vibe` 혼재, MyPage 비교표는 "바이브탭" |
