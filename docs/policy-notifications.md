# 알림 정책

> **구현 상태**: 알림 기능 미구현. 이 문서는 정책 확정 기록용.

## 1. 알림 트리거

| 이벤트 | 알림 발송 | 비고 |
|--------|----------|------|
| 평판 리뷰 작성됨 | ✅ | 프로필 주인에게 발송 |
| 피드백 요청 수신 | ✅ | 요청 받은 대상에게 발송 |
| 방명록 작성됨 | ✅ | 프로필 주인에게 발송 |
| 내 프로필 방문 | ❌ | 알림 미제공 (Pro 방문자 통계로 확인) |
| 가상 프로필 클레임 안내 | ❌ | 미구현 |

---

## 2. 알림 채널

알림은 **앱 내 알림**으로만 제공. 이메일·문자 발송 미지원.

---

## 3. 알림 설정

### 3-1. 세부 설정

알림 종류별 개별 on/off 설정은 **구현하지 않는다.**

### 3-2. 보관 기간

수신된 알림은 **최근 30일간 보관** 후 자동 삭제.

### 3-3. 읽음 상태 표시

| 상태 | 표시 방식 |
|------|---------|
| 읽지 않음 | 강조 배경 또는 점(dot) 표시 |
| 읽음 | 일반 배경, 강조 없음 |
