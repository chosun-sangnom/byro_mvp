# 평판 리뷰 정책

## 1. 작성 자격

### 1-1. 작성 가능 대상

작성 가능 여부는 프로필 주인의 **평판 탭 공개 범위 설정**에 따라 결정.

| 공개 범위 | 작성 가능 대상 |
|----------|--------------|
| 전체공개 (`public`) | 로그인 유저 누구나 / 비로그인 유저 (익명 처리) |
| 비공개 (`private`) | 작성 불가 (탭 잠금으로 진입 차단) |

*(~~연결된 사람만~~ — 연결 기능 제거로 삭제)*

### 1-2. 자기 자신에 대한 리뷰

프로필 주인은 **자기 자신에게 리뷰를 남길 수 없다.**

### 1-3. 중복 작성

동일인이 같은 프로필에 **여러 번 리뷰를 남길 수 있다.** (키워드 변경 포함) 별도 제재 없음.

---

## 2. 익명 정책

### 2-1. 익명 리뷰 작성

로그인한 사용자는 리뷰 작성 시 **익명으로 남기기**를 선택할 수 있다.

- 익명 작성 시 프로필 주인에게 작성자 정보 노출 안 됨
- **운영자도 익명 작성자 정보를 식별할 수 없다**
- DB에 IP 정보는 저장

> **TODO**: 익명 사용자 UI 표기 네이밍 확정 필요 (예: "익명", "누군가" 등)

### 2-2. 익명 리뷰 수신 거부 없음

익명 리뷰 수신 거부 설정은 **제공하지 않는다.** 익명으로 남기기 옵션은 항상 활성 상태이며, 공개 범위 설정과 무관하게 작성 가능한 사람은 누구나 익명을 선택할 수 있다.

---

## 3. 수정·삭제 정책

### 3-1. 작성자 수정·삭제

리뷰 작성자는 **수정 및 삭제 불가**. 기간 제한 없이 작성 후 변경 불가.

### 3-2. 프로필 주인의 리뷰 관리

프로필 주인은 자신의 프로필에 달린 특정 리뷰를 **숨기거나 삭제할 수 있다.**  
(편집 화면에서 처리, 구현 완료)

---

## 4. 악의적 리뷰 대응

### 4-1. 신고 주체

부적절한 리뷰 신고는 **프로필 주인만** 할 수 있다.

### 4-2. 신고 처리

신고된 리뷰는 **즉시 숨김 처리**된다. (운영자 검토 전 선제 차단)

### 4-3. 반복 악의적 작성 제재

허위·악의적 리뷰 반복 작성 계정 → 신고·제재 정책의 3단계 제재 적용.  
**경고 → 일시정지 → 영구정지**
